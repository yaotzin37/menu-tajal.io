# Tajal Restaurante & Mixología

Sitio web del menú del restaurante Tajal, especializado en sabores mexicanos con un toque de elegancia.

## Estructura del proyecto

- `index.html` - Página principal con el menú completo
- `videos/` - Carpeta para videos (video de fondo del hero)
- `images/platos/` - Imágenes de los platillos del menú
- `images/logo.png` - Logo del restaurante

## Cómo personalizar

1. Reemplaza las imágenes en la carpeta `images/platos/` con fotos reales de tus platillos
2. Modifica el video de fondo en `videos/hero-background.mp4`
3. Actualiza los precios y descripciones en el HTML según sea necesario
4. Personaliza los colores editando las variables CSS en la sección `:root`

## Publicación en GitHub Pages

1. Sube todos los archivos a un repositorio de GitHub
2. Ve a la configuración del repositorio > Pages
3. Selecciona la rama main como fuente
4. Tu sitio estará disponible en `https://[usuario].github.io/[nombre-repositorio]/`
